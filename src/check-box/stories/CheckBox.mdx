```tsx
export function Example() {
	const [checked, setChecked] = useState(false);

	return <CheckBox label="Согласен с условиями" value={checked} onChange={setChecked} />;
}
```

## Практические рекомендации

- Если состояние должно жить во внешней форме или модели, ориентируйтесь на контролируемый режим из документации.
- Сначала выберите семантику и визуальную схему, а затем уже подстраивайте остальные пропсы компонента.
- Проверьте размер компонента в реальном контексте интерфейса: в таблицах, тулбарах и формах требования обычно отличаются.
- Заблокированное состояние лучше проверять отдельно, чтобы убедиться, что визуально и поведенчески оно остается понятным.
