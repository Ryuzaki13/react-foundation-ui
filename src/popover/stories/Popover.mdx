## Основной сценарий

<Primary />

<Controls of={PopoverStories.Default} />

```tsx
import { Button, Popover } from "@/shared/ui";

export function Example() {
	return (
		<Popover>
			<Popover.Trigger>
				<Button>Открыть</Button>
			</Popover.Trigger>
			<Popover.Content>
				<div style={{ padding: 12 }}>Содержимое поповера</div>
			</Popover.Content>
		</Popover>
	);
}
```

## Практические рекомендации

- Если состояние должно жить во внешней форме или модели, ориентируйтесь на контролируемый режим из документации.
- Если компонент участвует в фильтрации или поиске, заранее определите дефолтные значения и допустимые ограничения.
