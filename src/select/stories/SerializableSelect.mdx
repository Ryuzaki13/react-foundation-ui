## Основной сценарий

<Primary />

<Controls of={SerializableSelectStories.Controlled} />

```tsx
import { useState } from "react";

import { SerializableSelect } from "@/shared/ui";

const options = [
	{ key: "draft", label: "Черновик" },
	{ key: "done", label: "Завершен" }
];

export function Example() {
	const [value, setValue] = useState(undefined);

	return <SerializableSelect label="Статус" value={value} onChange={setValue} options={options} optionKey="key" optionLabel="label" />;
}
```

## Практические рекомендации

- Если состояние должно жить во внешней форме или модели, ориентируйтесь на контролируемый режим из документации.
- Заблокированное состояние лучше проверять отдельно, чтобы убедиться, что визуально и поведенчески оно остается понятным.
