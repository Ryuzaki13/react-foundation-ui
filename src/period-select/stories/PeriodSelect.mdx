<Title>PeriodSelect</Title>
<Subtitle>Выбор периода детализации дат с ограничением вариантов по связанному диапазону</Subtitle>

PeriodSelect используется для выбора уровня детализации: `day`, `week`, `month` или `year`. Компонент хранит только id выбранного периода, а недоступность вариантов вычисляет по переданному `dateRange` и порогам `maxDayRangeDays` / `maxWeekRangeWeeks`.

Prop `presetIds` ограничивает встроенный каталог разрешенными вариантами. Независимо от порядка id в конфигурации контрол отображает их в каноническом порядке `day → week → month → year`. Отсутствующий, пустой или полностью невалидный список безопасно заменяется полным встроенным набором. Собственный `options` имеет приоритет над `presetIds`.

## Основной сценарий

<Primary />

<Controls of={PeriodSelectStories.Controlled} />

```tsx
import { useState } from "react";

import { PeriodSelect, type PeriodSelectPresetId } from "@ryuzaki13/react-foundation-ui/period-select";

export function Example() {
	const [value, setValue] = useState<PeriodSelectPresetId | undefined>("month");
	const dateRange = [new Date(2026, 2, 1), new Date(2026, 2, 14)] as const;

	return (
		<PeriodSelect
			label="Период"
			value={value}
			onChange={setValue}
			presetIds={["month", "year"]}
			dateRange={dateRange}
			maxDayRangeDays={31}
		/>
	);
}
```

## Практические рекомендации

- Передавайте `dateRange`, если период должен зависеть от выбранного диапазона дат.
- Используйте `presetIds`, когда runtime должен предлагать только разрешенные уровни детализации; порядок массива не управляет порядком Select.
- Настраивайте `maxDayRangeDays` и `maxWeekRangeWeeks` в конфиге параметрического контрола, а не в потребителях runtime.
- Если `dateRange` временно блокирует все разрешенные варианты, Select показывает пустое значение, но не очищает внешний controlled-id. После появления доступного варианта отображение восстанавливается без потери runtime-параметра.
- В story `Controlled` значения обновляются через `useArgs`, поэтому Controls и сам Select остаются синхронизированы.
